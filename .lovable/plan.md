
# 备份提示移至 Drawer 内部

## 变更概述

将主页面底部的"重要提示"卡片移除，并在云端备份和本地备份的 Drawer 中分别展示不同的提示内容。

## 具体改动

### 1. 删除主页面的提示卡片

移除第 285-298 行的 Tips Card 组件。

### 2. 云端备份 Drawer 添加提示

在密码表单下方添加云备份相关的提示：

| 提示内容 |
|---------|
| 备份将加密存储在您的 iCloud 账户中 |
| 备份密码无法找回，请务必牢记 |
| 换机时使用相同 Apple ID 登录即可恢复 |

### 3. 本地备份 Drawer 添加提示

在密码表单下方添加本地备份相关的提示：

| 提示内容 |
|---------|
| 备份文件将导出到您的设备 |
| 备份密码无法找回，请务必牢记 |
| 请将备份文件保存在安全位置（如电脑或 U 盘） |

### 4. 提示卡片样式

使用与主页面一致的警告样式：
- 背景色: `bg-amber-500/5`
- 边框: `border-amber-500/20`
- 图标: `AlertTriangle` (amber-500)
- 字体: 标题 `text-sm font-medium`，内容 `text-xs text-muted-foreground`
